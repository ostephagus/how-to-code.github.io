Converting types

Look at the table below, it tells you the syntax for converting types in python; put whatever you want to convert inside the brackets.

What you wat to convert to Syntax
String str()
Int int()
Float float()
List list()
Dictionary dict()

So, as you can see, it's all really easy to remember.
