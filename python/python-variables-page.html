Variables in Python

What are variables?

Variables are like digital envelopes or boxes; they can hold information. Although it doesn't sound like a useful thing to have yet, it's very important when you get more advanced.

Using variables in python

The syntax for creating a variable in python looks like this: variable_name = "hello". You can change variable_name to whatever you want your variable to be called and you can change "hello" to whatever you want although it must be in speech marks unless it is a number or the word True or the word False (with the capital letters). We will dig deeper into this in the types page.
