Python Types

What are types?

Types are the different ways of creating data. These are all shown in the table below:

Name of type What it's for Syntax Example
Integer Whole numbers only Just the number 5
Float Numbers with decimal points Just the number 5.0
Boolean The word True or False True or False with the capital letter True
String Every symbol on your keyboard In speech marks "Hello!"
List A list of all of the above types In square brackets and items are seperated by commas ["Buy dog food", 1234, True, 12.34]
Dictionary Phonebooks or anything where something means another thing In curly brackets, joined by colons and seperated by commas {"Emma": 0124539128, "Matt": 07434285121}

Floats and integers

Floats and integers (shortened to ints in python) are actually very similar, the only difference is that floats have a floating point (you may also know this as decimal point) and integers don't.

Booleans

Booleans are the type that python uses the most. This will make more sense in the Python conditions and condition statements page but basically, if you ask Python a question, it will reply with True or False instead of the everyday yes or no.

Strings

Strings are definitely one of the most useful types in python because they are the easiest for humans to understand.

Lists

Lists are really self explanatory: they're good for lists. The good thing about lists is that they can hold any type, including other lists!

Dictionaries

Dictionaries are by far the hardest type to get to grips with. They're confusing, seemingly useless and in fact aren't often used in computing. One really vital use for them is when you want to log in to a webpage. The username can be matched up with the password or other things like that

The first part of a dictionary is called the key. It is used to reference whatever is at the other side of the colon to it which is caled the value.
